# Soluble-Factors Excel data parser

Data are supplied in wide-format, Excel files across several
worksheets. This R-script, `parse-Trittel-Soluble-Stim.R`, parses that
Excel file into a long-format CSV file that uses the controlled
vocabulary developed for this INCENTIVE project. See the
[Controlled-Vocabulary repo](https://github.com/INCENTIVE-Grant/Controlled-Vocaulary)
in this same INCENTIVE-Grant project.

The log file from the parsing is also stored here. See the file
`parse-Trittel-Soluble-Stim_20260312.log`.

The data files, both Excel and CSV, associated with this parser are located on Zenodo:
https://doi.org/10.5281/zenodo.15363668 .
