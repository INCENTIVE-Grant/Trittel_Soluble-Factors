# Soluble-Factors Excel data parser
Data are supplied in wide-format, Excel files across several worksheets. This R-script, `parse-Trittel-Soluble-Stim.R`, parses that Excel file into a long-format CSV file that uses the controlled vocabulary developed for this INCENTIVE project. See the Controlled-Vocabulary repo in this same INCENTIVE-Grant project.

The log file from the parsing is also stored here. See the file `parse-Trittel-Soluble-Stim_20260309.log`.
